<screen-saver>

A vanilla JavaScript web component that displays retro screen saver effects after idle timeout. Bring back the nostalgia!

View Demos

Features

  • Zero dependencies - pure vanilla JavaScript
  • Three built-in effects: bounce3d, matrix, ascii-glitch
  • Automatic idle detection with configurable timeout
  • Custom effect registration API
  • Full CSS custom property support
  • Respects prefers-reduced-motion
  • TypeScript definitions included

Quick Start

<!-- Add to your HTML -->
<script type="module" src="screen-saver.js"></script>

<screen-saver timeout="180" effect="bounce3d">
  Your Text Here
</screen-saver>

Installation

npm

npm install screen-saver

CDN

<script type="module" src="https://unpkg.com/screen-saver/dist/screen-saver.js"></script>

ES Module

import { ScreenSaver } from 'screen-saver';

// Use programmatically
const el = document.querySelector('screen-saver');
el.activate();

Available Effects

bounce3d

Classic bouncing 3D extruded text with color cycling on edge bounces. Features perspective rotation and layered text shadows.

matrix

Matrix-style falling character rain with Japanese katakana and ASCII. Your text glows prominently in the center.

ascii-glitch

Glitchy ASCII effect with random character corruption, chromatic aberration layers, and flicker effects.

Built with vanilla JavaScript. No dependencies.
