Advanced Effect Configuration

Advanced effects support data attributes for configuration:

<!-- Slideshow with custom images --> <screen-saver effect="slideshow" data-images="photo1.jpg,photo2.jpg,photo3.jpg" data-duration="5000" data-transition="1000"> My Photos </screen-saver> <!-- Snow with custom background --> <screen-saver effect="snow" data-background="winter-cabin.jpg" data-snowflakes="300" data-wind="0.5"> Let it Snow </screen-saver>

CSS Customization

Customize the look with CSS custom properties:

screen-saver { --screen-saver-bg: rgba(0, 0, 0, 0.95); --screen-saver-text-color: #00ff00; --screen-saver-font-family: 'Arial Black', sans-serif; --screen-saver-font-size: 4rem; --screen-saver-z-index: 999999; }

Reduced Motion

The component automatically respects prefers-reduced-motion. When enabled, all effects display static centered text without animation.

To test, enable "Reduce motion" in your system accessibility settings.

Live Slot Updates

Click the button to push a live clock into the slot while the screen saver is active.

Move mouse or press any key to dismiss the screen saver.
