CSS Customization

Customize the look with CSS custom properties:

screen-saver {
  --screen-saver-bg: rgba(0, 0, 0, 0.95);
  --screen-saver-text-color: #00ff00;
  --screen-saver-font-family: 'Arial Black', sans-serif;
  --screen-saver-font-size: 4rem;
  --screen-saver-z-index: 999999;
}

Reduced Motion

The component automatically respects prefers-reduced-motion. When enabled, all effects display static centered text without animation.

To test, enable "Reduce motion" in your system accessibility settings.

Move mouse or press any key to dismiss the screen saver.
